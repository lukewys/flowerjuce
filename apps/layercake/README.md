# layercake

![layercake](/assets/layercake.png)

layercake is a granular sound design playground with deeply integrated modulation. it combines 8 channels of granular synthesis with 8 polymorphic LFOs (inspired by mutable instruments' stages/pamela's new workout).

## keyboard controls

layercake is designed to be playable without a mouse.

| Key | Action |
| --- | --- |
| `?` | Show keyboard shortcuts overlay |
| `Space` | Open command palette (fuzzy search controls) |
| `Tab` / `Shift+Tab` | Cycle focus between controls |
| `Arrows` | Adjust focused parameter |
| `Shift` + `Arrows` | Coarse adjustment |
| `Alt` + `Arrows` | Fine adjustment |
| `Enter` | Edit value (type number) |
| `l` + `1`-`8` | Focus LFO 1-8 |
| `t` | Tap tempo (repeated presses) |
| `r` | Toggle record |
| `g` then `r` | Randomize focused parameter section |
